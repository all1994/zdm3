1. pdf-lib-master_installed.zip
Open source of pdf-lib.
npm modules are installed.
This is used if you want to modify pdf-lib.

2. pdf-lib-docs-master_installed.rar
Document of pdf-lib.
npm modules are installed.
You can run document site by "npm start" in root directory or website subdirectory.

3.pdf-lib.js, pdf-lib.min.js
Distributed library of pdf-lib.
Just include one if it in your html and use it as the example in doc.
